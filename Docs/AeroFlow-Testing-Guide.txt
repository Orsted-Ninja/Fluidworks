================================================================================
 1. APPLICATION OVERVIEW
================================================================================

AeroFlow is a real-time Computational Fluid Dynamics (CFD) application built in
Unity 6000.x with the Universal Render Pipeline (URP). It provides four core
simulation modes accessible via the Home tab ribbon:

  - External Aero    : Wind tunnel analysis (Navier-Stokes grid solver)
  - Internal Flow    : Pipe/duct flow analysis (pressure drop, friction)
  - Rotating Machinery: Fan/turbine/impeller with MRF rotating zone
  - Free-Surface Flow: Dam break / sloshing (SPH particle method)

Key outputs: drag/lift coefficients, pressure maps, streamlines, torque,
efficiency, head loss, splash dynamics, and CSV time-series export.

================================================================================
 2. GETTING STARTED
================================================================================

LAUNCHING:
  1. Open the project in Unity 6000.3.1f1 or later.
  2. Open scene: Assets/Scenes/SampleScene.unity (main entry point).
  3. Enter Play Mode (Ctrl+P or the Play button).
  4. The application UI loads with Home, Simulation, and View tabs.

LOADING A SIMULATION:
  1. Click one of the template buttons on the Home tab:
       - "External Aero"       -> Wind tunnel with model loading
       - "Internal Flow"       -> Pipe flow simulation
       - "Rotating Machinery"  -> Fan/turbine with rotation zone
       - "Free-Surface Flow"   -> Dam break / SPH

  2. Wait for the scene to load (progress bar appears).
  3. Load a 3D model via the "Load Model" button in the properties panel
     (supports .obj, .stl, .fbx, .glb, .gltf).
  4. Click "Play" on the simulation tab or the quick-play button.

  Speed 80:  Re ~500k-2M,       Cd ~0.2-0.6

---- Test 4.5: Custom Wind Direction ----

---- Test 4.6: Turbulence Stress Test ----

Test turbulence levels: 1, 10, 25, 40

Expected:

---- Test 4.7: Metrics Validation (Physics Check) ----

For a bluff body at 50 m/s in air (rho=1.225, mu=1.81e-5):
  - Re = rho * V * L / mu. For L=1m: Re = 1.225 * 50 / 1.81e-5 ~ 3.4M
  - Drag force = 0.5 * rho * V^2 * A * Cd
  - For Cd=0.5, A=1.0: F_drag = 0.5 * 1.225 * 2500 * 1.0 * 0.5 = 766 N

Verify that displayed metrics are physically reasonable.

 5. TEST SUITE: INTERNAL FLOW (PIPE FLOW)
================================================================================

PREREQUISITE: Select "Internal Flow" template, load a pipe/duct model
(optional - works without model too), click Play.

---- Test 5.1: Laminar Pipe Flow ----

Input Values:
  Pipe Radius:          0.05 m  (50 mm)
  Pipe Length:          2.0 m
  Inlet Velocity:       0.5 m/s
  Fluid Density:        998 kg/m3 (water)
  Dynamic Viscosity:    1.002e-3 Pa.s (water at 20C)
  Laminar:              ON (checked)
  Wall Roughness:       0.0015 m
  Time Scale:           1.0
  Iterations/Frame:     4

Expected Results (analytical validation):
  Pipe Re = rho * V * D / mu = 998 * 0.5 * 0.1 / 0.001 = 49,900
  NOTE: With Laminar forced ON, solver uses f = 64/Re regardless
  Friction Factor f = 64 / 49900 = 0.00128
  Head Loss hL = f * (L/D) * (V^2 / 2g) = 0.00128 * 20 * 0.01274 = 0.000326 m
  Flow Rate Q = V * pi * r^2 = 0.5 * pi * 0.0025 = 0.00393 m3/s
  Pressure Gradient = dP/L

Pass Criteria:
  [x] Friction factor is displayed and > 0
  [x] Head loss is displayed and > 0
  [x] Flow rate ~ pi * r^2 * V_mean
  [x] Reynolds number matches expected order of magnitude

---- Test 5.2: Turbulent Pipe Flow ----

Input Values:
  Pipe Radius:          0.5 m
  Pipe Length:          5.0 m
  Inlet Velocity:       10 m/s
  Fluid Density:        1.225 kg/m3 (air)
  Dynamic Viscosity:    1.81e-5 Pa.s (air)
  Laminar:              OFF
  Wall Roughness:       0.0015 m
  Time Scale:           1.0
  Iterations/Frame:     4

Expected Results:
  Pipe Re = 1.225 * 10 * 1.0 / 1.81e-5 ~ 676,796
  This is fully turbulent (Re > 4000)
  Colebrook-White: e/D = 0.0015, f ~ 0.02-0.03 (typical for rough pipe)
  Head Loss hL = f * (L/D) * V^2 / 2g ~ f * 5 * 100/19.62 ~ f * 25.5

  For f = 0.025: hL ~ 0.638 m
  Flow Rate Q = 10 * pi * 0.25 = 7.854 m3/s

Pass Criteria:
  [x] Friction factor is in range 0.01 - 0.06
  [x] Reynolds number > 4000 (turbulent regime)
  [x] Pressure drop is positive
  [x] Wall shear stress is positive
  [x] Head loss is reasonable (not NaN or extremely large)

---- Test 5.3: Transition Check ----

Gradually increase inlet velocity from 0.1 to 5 m/s with:
  Pipe Radius: 0.05m, Density: 1.225, Viscosity: 1.81e-5

Re thresholds:
  Re < 2300:  Laminar    -> f = 64/Re
  Re > 4000:  Turbulent  -> f = Colebrook-White
  2300-4000:  Transition  -> should use turbulent formula

Pass Criteria:
  [x] Friction factor changes at Re ~ 2300
  [x] No discontinuity crash at transition

---- Test 5.4: Edge Cases ----

  - Very low velocity (0.01 m/s): should not crash, Re very low
  - Very high velocity (100 m/s): should remain stable
  - Very small pipe radius (0.001 m): extreme Re, check for NaN
  - Zero viscosity: should clamp to minimum, not divide by zero

 9. VIDEO EXPORT VERIFICATION
================================================================================

HOW TO EXPORT:
  1. Run any simulation.
  2. Click "Export Video (MP4)" on the Simulation tab.
  3. Choose a save location for the .mp4 file.
  4. Recording starts immediately. Click "Stop Recording" to finish.

BEHAVIOR:
  - Captures ONLY the simulation camera (excludes UI overlay).
  - Uses Camera.Render() to a RenderTexture at 1920x1080, 30 FPS.
  - Maximum duration: 30 seconds (auto-stops).
  - Frames saved as PNG sequence, then encoded to MP4 via ffmpeg.
  - PNG frames are deleted after successful encoding.

VALIDATION STEPS:
  1. Verify the button text changes to "Stop Recording" while recording.
  2. Verify the button returns to "Export Video (MP4)" after stopping.
  3. Check that the .mp4 file is created at the chosen location.
  4. Open the .mp4 and verify:
     [x] Video shows the simulation (particles, streamlines, etc.)
     [x] Video does NOT show the UI (ribbon, panels, buttons)
     [x] Video resolution is 1920x1080
     [x] Video duration matches recording time
  5. If ffmpeg is not installed, verify:
     [x] Warning appears in console: "ffmpeg.exe not found"
     [x] PNG frames are saved in AppData/.../Captures/ folder

FFMPEG LOCATION SEARCH ORDER:
  1. ffmpegPathOverride field (if set)
  2. StreamingAssets/Tools/ffmpeg.exe
  3. System PATH

 12. KNOWN LIMITATIONS
================================================================================

  - Pipe Flow and Rotating Machinery require compute shader files
    (PipeFlow3D.compute, RotatingMachinery3D.compute) in the
    Resources/Compute/ folders. If missing, the solver logs an error
    and does not run.

  - FSI Lite and Playback/Validation templates are placeholders.
    They load the wind tunnel scene but do not have dedicated solvers.

  - Video export requires ffmpeg.exe. Without it, only PNG frame
    sequences are saved.

  - The solver grid resolution is limited by GPU memory. Very high
    resolutions (128x128x128) may cause performance issues on
    integrated GPUs.

  - Model loading supports .obj, .stl, .fbx, .glb, .gltf formats.
    Very large models (>500k triangles) may slow voxelization.

  - CSV export writes all 46+ columns regardless of mode. Columns
    not relevant to the current mode will show zero values.

QUICK REFERENCE: PHYSICAL CONSTANTS FOR TESTING

Fluid          | Density (kg/m3) | Viscosity (Pa.s)  | Notes
---------------|-----------------|-------------------|------------------
Air (20C)      | 1.225           | 1.81e-5           | Default
Water (20C)    | 998             | 1.002e-3          | For pipe flow
SAE 30 Oil     | 880             | 0.29              | High viscosity
Glycerin       | 1260            | 1.412             | Very high visc.

REYNOLDS NUMBER THRESHOLDS:
  Re < 2300   : Laminar flow
  2300 < Re < 4000 : Transitional
  Re > 4000   : Turbulent

KEY FORMULAS:
  Re = rho * V * L / mu              (Reynolds number)
  Cd = Fd / (0.5 * rho * V^2 * A)    (Drag coefficient)
  Cl = Fl / (0.5 * rho * V^2 * A)    (Lift coefficient)
  f = 64/Re                          (Laminar friction factor)
  hL = f * (L/D) * V^2 / (2*g)       (Darcy-Weisbach head loss)
  Q = V * A                          (Volumetric flow rate)
  T = dP * Q / omega                 (Torque estimate)
  P = T * omega                      (Shaft power)
  Re_machine = omega * R^2 / nu      (Machine Reynolds)
